```

```

## Mongo
###### Installation
+ Debian-based distribution like Ubuntu
```sh
$ sudo apt-get install mongodb
```
+ Mac
```sh
$ brew install mongodb
```
[Documentation](https://docs.mongodb.com/manual/introduction)
